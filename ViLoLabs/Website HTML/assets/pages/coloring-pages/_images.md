TWO PROMPT STYLES IN THIS FILE:
  · SINGLE-SUBJECT (rows 1-100) — one creature/object filling the page.

    (forest, farmhouse, beach, etc.) Best for older children (Class 1-5)
    who can colour multiple regions and pick their own palette per area.

REFERENCE THUMBNAIL — IMPORTANT CHANGE (2026-05-26):
  Earlier prompts said "small fully colored reference thumbnail" in the
  corner. That made the reference too tiny to read distinct colours on.
  ALL prompts now say "LARGER reference thumbnail (about 25 percent of
  the page width, big enough to see every distinct region in its final
  colour clearly)". This matters most for the new SCENE prompts where
  kids need to see what colour each region (tree trunks vs leaves vs
  sky vs path) should be. Apply the same change if you regenerate
  existing pages.
