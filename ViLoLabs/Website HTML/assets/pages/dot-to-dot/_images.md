# Dot-to-Dot Pictures — Image List (100)

ASSET TYPE: HALF-FINISHED dot-to-dot worksheet (line + dots baked in).
SPEC: JPG or PNG, A4 portrait, 2480x3508 px (300 DPI).
Each image is split in half — ONE half is drawn as a complete bold
continuous outline (so the child sees what they are making), the OTHER
half is replaced with a sequence of small numbered dots along where the
outline would go. Connecting the dots 1→2→3→… in order completes the
picture. Dots and numbers are intentionally SMALL so the visible line
half stays the visual focus.

NAMING: entry names have NO file extension. Save as .jpg OR .png.
