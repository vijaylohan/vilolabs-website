Good to Know

Common questions.

Are these tools really free?

Yes — completely free, with no sign up and no watermark on your output. Use them as often as you like.

Are my files uploaded to a server?

No. Each tool runs entirely inside your web browser. Your files never leave your device, so it is fully private and safe.

Do the tools work on mobile?

Yes — every tool works on phones, tablets and computers in any modern browser.

Which tool should I open first?

For documents, try Image to PDF or Merge PDF. For photos, try Resize Image (it has a 200 KB preset for Indian govt forms).
