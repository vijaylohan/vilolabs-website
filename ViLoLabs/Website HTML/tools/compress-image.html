Good to Know

Common questions.

What quality should I pick?

75% is a good default for most photos — you save a lot of size without seeing a difference. Drop to 60% for email attachments, push to 85%+ for important detail.

Why doesn't PNG get much smaller?

PNG is a lossless format — the slider doesn't apply to it the way it does to JPG/WebP. To compress PNG more, switch the output format to JPG or WebP.

What's WebP good for?

WebP is a modern format that's typically 25–35% smaller than JPG at the same visual quality. Great for the web, supported in all current browsers.

Need exact file size like 200 KB?

Use Resize Image instead — it has a target-KB mode with a 200 KB preset for govt forms.
